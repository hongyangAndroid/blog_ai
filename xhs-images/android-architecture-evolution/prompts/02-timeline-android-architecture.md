# 02 Timeline - 7个阶段完整对比

## 基础设定
- **比例**: 3:4 (小红书标准)
- **风格**: Notion极简手绘风
- **背景**: 白色/浅灰，带淡淡网格

## 画面描述
创建一个信息密度适中的技术时间线图：

**标题**（顶部）：
"7个阶段完整演进"

**内容布局**（垂直排列7个卡片）：

每个阶段一个横向卡片，包含：
1. **左侧**：年份区间（如"2010-2013"）
2. **中间**：技术名称（如"无架构"）
3. **右侧**：核心特点（简短1-2词）

**7个阶段内容**：

📱 Phase 1 | 2010-2013 | 无架构 | 代码堆砌
📱 Phase 2 | 2013-2016 | MVP | 关注点分离
📱 Phase 3 | 2014-2017 | Clean+Dagger | 业务解耦
📱 Phase 4 | 2016-2019 | MVVM+AAC | 生命周期感知
📱 Phase 5 | 2017-2020 | RxJava | 异步简洁
📱 Phase 6 | 2019-2022 | Coroutines+Flow | 背压友好
📱 Phase 7 | 2020-2025 | MVI+Compose | 声明式UI

**视觉设计**：
- 每个卡片用细边框分隔
- 阶段编号用圆形背景突出
- 技术名称用稍大字体
- 整体从左到右有淡淡的渐变箭头暗示演进方向
- 配色：深蓝标题 + 灰色正文 + 橙色高亮关键词

**装饰**：
- 顶部有细线分隔标题
- 底部有淡淡的"进化"箭头装饰

## 风格关键词
Notion style, minimalist, information card, timeline, clean layout, grid system, technical documentation, blue and gray color scheme, educational infographic, modern UI
