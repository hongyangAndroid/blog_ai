# 03 Guide - 技术选型指南

## 基础设定
- **比例**: 3:4 (小红书标准)
- **风格**: Notion极简手绘风
- **背景**: 白色，四象限布局

## 画面描述
创建一个四象限技术选型指南图：

**标题**（顶部居中）：
"不同场景该选什么架构？"

**四象限布局**（2x2网格）：

**左上象限 - 小型Demo** 🎯
- 标题：小型Demo/原型
- 推荐：无架构 或 MVP
- 原因：快速开发，验证想法
- 图标：闪电/火箭

**右上象限 - 中小型App** 📱
- 标题：中小型App
- 推荐：MVVM + AAC
- 原因：生命周期感知，社区成熟
- 图标：手机

**左下象限 - 中大型App** 🏢
- 标题：中大型App
- 推荐：Clean + Coroutines
- 原因：业务解耦，可维护性强
- 图标：建筑/分层

**右下象限 - 大型复杂App** 🚀
- 标题：大型复杂App
- 推荐：MVI + Compose
- 原因：状态可预测，声明式UI
- 图标：火箭/齿轮

**视觉设计**：
- 四个象限用细线分隔
- 每个象限有淡色背景区分（浅蓝、浅绿、浅黄、浅橙）
- 推荐技术用粗体+橙色高亮
- 图标简洁线条风格

**底部提示**（小字）：
"根据项目规模选择合适的架构"

## 风格关键词
Notion style, quadrant layout, decision matrix, minimalist icons, soft pastel backgrounds, clean typography, technical guide, infographic, modern design, educational
